1. 음성 -> string
- 버튼을 사용하여 누른 경우에만 음성 입력
- 사용할 모듈은 일단 google

2. 질문 만들기
- 우리 회의때마다 무조건 10개 이상은 만들기

3. 유사도 비교를 통해 질문 선정
- 진모가 보여준 모듈 찾아서 예제로 써보기
- 유사도 끼리 서로 조합 (voting, meaning)
- 출력된 확률 분포를 통해 질문 설정

(
3-1. 3이 잘 안돌아가는 경우
- RNN, LSTM 모델 구현
)

4. 출력 단어 구성
- 미리 만들어 두기 (일단은 문장 형태)
- 질문과 답변을 일련번호로 연결 
          (답변은 나중에 class 형태로 변경하여 유동적으로 변경)

(
4-1, 5. 답변에서 중요 정보 불러오는 과정
- 답변은 질문을 포함하는 형태로 만든다.
- 질문에서 명사를 가져와서, 해당 명사에 맞는 정보를 가져온다.
- 위에서 가져온 정보를 통해 string 을 만든다.
)
> 정보 가져오는 구현에서 한번 해보기

6. 문장 출력 모듈
- TTS 모듈 찾아서 써보기

- 음성 입력, 문장 출력 이번주 주말 전까지 (~1/3)
